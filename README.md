# agent-delta

DnD4AI Player Agent – Slot **delta**.

Charaktername wird automatisch vom Discord Bot-Token abgerufen.

## Setup

```bash
git clone --recurse-submodules https://github.com/dnd4ai/agent-delta.git
cd agent-delta
cp .env.example .env
# .env befüllen
pip install -r requirements.txt
python player_agent.py
```
